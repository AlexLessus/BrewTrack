I recently started diving into the world of specialty coffee and realized I needed a consistent way to track my brews. The easy way is to buy a notebook, but I saw an opportunity to learn something new, I wanted to build a digital tool that offers precision and data persistence.

This project is also a personal journey to master **Modern Android Development** (MAD), moving away from legacy XML layouts to a fully declarative UI with Jetpack Compose.
